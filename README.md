# Android-приложение "Заметки"

Учебный проект для практики по мобильной разработке. Приложение позволяет создавать заметки, указывать срок выполнения, просматривать список, редактировать и удалять записи.

## Тема

Приложение для заметок со сроком выполнения.

## Основные функции

- просмотр списка заметок;
- добавление новой заметки;
- просмотр подробной информации;
- редактирование заметки;
- удаление заметки;
- отметка "Выполнено";
- сохранение данных после перезапуска приложения.

## Экраны

1. Главный экран со списком заметок.
2. Экран добавления новой заметки.
3. Экран просмотра и редактирования заметки.

## Используемые технологии

- Java;
- XML-разметка;
- Activity;
- RecyclerView;
- SQLite.

## Хранение данных

Для хранения используется локальная база SQLite. Таблица `notes` содержит поля:

- `id` - идентификатор записи;
- `title` - название заметки;
- `text` - текст заметки;
- `deadline` - срок выполнения;
- `done` - статус выполнения.

## Структура проекта

- `MainActivity.java` - главный экран и список заметок;
- `AddNoteActivity.java` - добавление новой заметки;
- `NoteDetailActivity.java` - просмотр, редактирование и удаление;
- `Note.java` - модель данных;
- `NotesDbHelper.java` - работа с SQLite;
- `NotesAdapter.java` - адаптер для RecyclerView.

## Проверка ввода

Перед сохранением проверяется название заметки. Если поле пустое, приложение показывает ошибку и не сохраняет запись.

## Инструкция по запуску

1. Открыть проект в Android Studio.
2. Дождаться синхронизации Gradle.
3. Запустить приложение на эмуляторе или Android-устройстве.
4. Нажать кнопку `+`, добавить заметку и проверить список.
